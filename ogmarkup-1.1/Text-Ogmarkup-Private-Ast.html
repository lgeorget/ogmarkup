data Atom a

An Atom is the atomic component of a Ogmarkup document. It can be either a punctuation mark or a word, that is a string.

Note that, by construction, OpenQuote and CloseQuote are not valid Mark values here. Indeed, they are implicit with the Quote constructor. This choice allows to enforce that an opened quote needs to be closed.
